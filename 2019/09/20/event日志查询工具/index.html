event日志查询工具

Posted on 2019-09-20 | Post modified: 2019-10-03

尝试做了个日志查询平台，用来查询来自elasticsearch的k8s event日志。

简介

  • 前端：vue+elementui，用来呈现event数据，并做了一定的分类，排序。
  • 后端：flask，用来获取elasticsearch的数据，并提供接口供前端调用。
  • 程序间数据交换格式均为json。

实现代码

es_client.py

es_client.py作为http客户端，向elasticsearch发送请求以获取event日志。http请求使用python的requests库即可，但主要难点在于构造请求包，通过日期，appId，clusterId等条件筛选结果。

web.py

web.py是工具的主体，一方面提供前端界面，另一方面则负责提供API。

由于使用ajax通信，flask无需再渲染模板，发送index.html静态文件即可。

index.html

index.html提供了前端界面，主要为请求的表单以及作为返回结果的表格。

小结

这个event日志查询工具是为了更加方便的查询elasticsearch中的数据，另外可搭建kibana查询，其更为通用些，却略为复杂。

该工具也锻炼了我的web应用编写能力，在写程序时小问题不断，比如说es查询语法错误、时间格式不符、ajax变量作用域、html样式不好看等等，最终一一解决。
